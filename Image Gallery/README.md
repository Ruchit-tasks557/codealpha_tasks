# CodeAlpha Internship - Interactive Image Gallery

A visually appealing, fully responsive, and interactive Image Gallery web application built during my Web Development Internship at CodeAlpha. This project focuses on modern UI layout designs and seamless interactive elements to provide an engaging user experience for viewing images.

## 🚀 Live Demo
Check out the live deployment of the project here:
👉 [Live Demo Link](https://ruchit-tasks557.github.io/codealpha_tasks/Image%20Gallery/)

## 📌 Features
* **Grid Layout:** Beautifully structured grid system showcasing images neatly across the page.
* **Interactive Lightbox/Preview:** Click on any thumbnail image to open it in a smooth, full-screen lightbox or modal overlay for a detailed view.
* **Navigation Controls:** Next and Previous buttons (or intuitive navigation) to browse seamlessly through the gallery images within the preview modal.
* **Responsive Design:** Completely adaptive layout utilizing CSS media queries to ensure a perfect look on smartphones, tablets, and desktop monitors.
* **Smooth Transitions:** Polished hover effects and modal fade-ins to enhance user engagement.

## 🛠️ Tech Stack
* **HTML5:** Structured the gallery elements, thumbnails, and preview modal markup.
* **CSS3:** Designed the visual aesthetic, grid alignments, hover effects, overlays, and responsive breakpoints.
* **JavaScript (ES6):** Developed the interactive functionality for opening/closing the lightbox and cycling through images.
