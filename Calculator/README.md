CodeAlpha Internship - Basic Calculator
A sleek, responsive, and fully functional Basic Calculator web application built during my Web Development Internship at CodeAlpha. This project demonstrates the practical application of core front-end technologies to create an interactive and user-friendly calculator interface.

Check out the live deployment of the project here:

📌 Features
Core Arithmetic Operations: Supports addition, subtraction, multiplication, and division.
Real-time Display: Instantly updates the screen as buttons are clicked.
Clear & Delete Functions: Includes an AC (All Clear) button to reset the calculator and a DEL button to remove the last entered digit.
Responsive Design: Fully optimized for all screen sizes, including mobile phones, tablets, and desktops.
Modern UI: Clean aesthetic featuring smooth hover effects, shadow depth, and a focused layout.
🛠️ Tech Stack
HTML5: Structured the layout and semantic buttons of the calculator.
CSS3: Styled the interface with a modern look, grid layouts, and interactive transitions.
JavaScript (ES6): Implemented the logic for mathematical evaluations and button click operations.
